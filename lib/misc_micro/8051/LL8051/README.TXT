Files of the sub-directory \SOURCE\MICRO\8051\TR8051

  This project is a TMEX example program written in 8051 Assembly.
  It is a Stand-alone piece of software implementing all functions 
  up to the Transport Layer to exchange data with a Touch Memory, 
  using the code of LL8051.ASM. Includes a 16-bit CRC calculation 
  example. Not designed for use with NW8051.ASM !

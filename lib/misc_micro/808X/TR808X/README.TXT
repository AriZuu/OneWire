Files of the sub-directory \SOURCE\MICRO\808x\TR808x

  This project is a TMEX example program written in 808x Assembly.
  It is a Stand-alone piece of software implementing all functions 
  up to the Transport Layer to exchange data with a Touch Memory, 
  using the code of LL8088.ASM. Includes a 16-bit CRC calculation 
  example. Not designed for use with NW8086.ASM !
